🔎 Introducción

La API de Los Simpson permite obtener información de personajes, incluyendo su nombre, género, estado, ocupación, imagen e historia. Es gratuita y pública.

📌 Endpoint principal

https://apisimpsons.fly.dev/api/personajes

Este endpoint devuelve un listado de personajes.

Parámetros opcionales:

  • limit: cantidad de resultados por página (default: 10).
  • page: número de página (default: 1).
  • nombre: filtra personajes por nombre exacto.

Ejemplo de solicitud:

GET https://apisimpsons.fly.dev/api/personajes?limit=10&page=1

.then(response => response.json())
.then(data => {

.catch(error => {
